Frequently asked questions

How do I block websites on Mac?

Install AppBlocker with the one-line terminal command above, then run appblocker block youtube.com (or any domain). It modifies your system hosts file to block the site at the OS level — no browser extension needed. Works across all browsers. Full guide here.

How do I block apps on Mac?

Run appblocker block "AppName" in Terminal. AppBlocker monitors running processes and immediately terminates blocked apps when they launch. Works with any Mac application. Full guide here.

How to block YouTube on Mac?

Run appblocker block youtube.com to block the website in all browsers, and appblocker block "YouTube" to block the YouTube app. Both at once. Done. Full guide here.

Is AppBlocker free?

Yes. Completely free. Open source under the MIT license. No trial, no freemium, no credit card. The entire tool is 60 lines of Swift. See the source on GitHub.

Does AppBlocker collect my data?

No. Zero telemetry. Zero analytics. Zero network requests. AppBlocker runs entirely on your machine. You can read every line of code — there are only 60 of them.

How do I unblock an app or website?

Run appblocker unblock "AppName" or appblocker unblock youtube.com. Instant. No cooldown period (unless you want one — that's on you).
