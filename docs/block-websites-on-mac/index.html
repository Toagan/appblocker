How to Block Websites on Mac (Free, No App Required)

Browser extensions are a joke. You install them, you disable them, you reinstall them. Here's something that actually works: block websites at the system level so no browser can reach them.

curl -fsSL https://raw.githubusercontent.com/Toagan/appblocker/main/install.sh | bash

Step-by-step: Block any website on Mac

  1. Open Terminal

    Press Cmd + Space, type "Terminal", hit Enter. That's where the magic happens.

  2. Install AppBlocker

    Paste this command and press Enter:

    curl -fsSL https://raw.githubusercontent.com/Toagan/appblocker/main/install.sh | bash

  3. Block the website

    Tell AppBlocker what to block. Use the domain name:

    appblocker block youtube.com

  4. Verify it's blocked

    Open any browser. Go to the site. It won't load. Across Chrome, Safari, Firefox, Arc — all of them. Because the block is at the system level, not the browser level.

Why this beats browser extensions

Browser extensions have one fatal flaw: you can disable them in 2 seconds. The moment your willpower dips, you click "Disable extension" and you're back on Reddit.

AppBlocker works at the system level. It modifies your hosts file. That means:

  • Works across all browsers simultaneously
  • Can't be bypassed by switching to another browser
  • Can't be disabled by right-clicking an extension icon
  • Works even in incognito / private browsing mode

You have to deliberately go back to Terminal and run the unblock command. That friction is the point. It gives you just enough pause to ask: "Do I really need this right now?"

FAQ: Blocking websites on Mac

Can I block a website on Mac without installing software?

Technically you could edit /etc/hosts manually, but it's tedious and error-prone. AppBlocker automates this with a single command and doesn't require a full app installation — it's a lightweight script.

Does this work on Safari, Chrome, and Firefox?

Yes. All of them. And Arc, Brave, Edge, Opera — every browser. Because the blocking happens at the operating system level, not inside any specific browser.

Can I block a website during specific hours only?

Scheduled blocking is coming soon. For now, you can combine AppBlocker with a cron job to block and unblock on a schedule. Or just block it and keep it blocked — you'll survive.

How do I unblock a website?

Run appblocker unblock youtube.com in Terminal. Instant. No cooldown, no waiting period.
