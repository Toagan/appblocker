FAQ for remote workers

Can I use this on my company Mac?

If you have Terminal access and admin privileges, yes. AppBlocker is a lightweight script — no enterprise software to install. It doesn't modify system settings beyond /etc/hosts for website blocking. If your company uses MDM that restricts terminal access, you may need to ask IT.

Will my employer see what I've blocked?

No. AppBlocker runs locally, stores data locally, and makes zero network requests. Your employer has no way to know you're using it — unless you tell them (and they'll probably be impressed).

Can I block Slack during focus time but keep it for meetings?

Yes. Block Slack when you need deep focus, unblock it for meetings. It takes 2 seconds: appblocker unblock "Slack". You can even set up a cron job to auto-unblock at meeting times.

What about blocking after work hours?

Great idea. Protect your personal time by blocking work apps after hours. Run appblocker block "Slack" at 6pm, unblock at 9am. Set it up with cron to automate it. Work-life boundaries, enforced by code.
